# Portfólio Dev

Este projeto é uma página de portfólio pessoal desenvolvida para apresentar o perfil profissional de **Felipe Mendes**, com foco em desenvolvimento **Full Stack**.

A proposta é  atravez de uma interface moderna e objetiva, apresentar uma introdução profissional, projetos em destaque, serviços oferecidos e canais de contato.

## Tecnologias utilizadas

- HTML5
- CSS3

## Objetivo

Servir como um portfólio profissional responsivo, de fácil manutenção e com layout pronto para evolução com novos projetos e integrações.
